## 記録様式

| 項目 | 内容 |
|---|---|
| ID | 連番（S-001, S-002, ...） |
| 資料名 | 公式資料名（原文ママ） |
| 取得元URL | ダウンロードした正確なURL |
| 出典ページ | 資料が掲載されているページのURL |
| 発行主体 | 省庁・部会名 |
| 取得日時 | ローカル取得日時（タイムゾーン付き） |
| SHA-256 | 取得ファイルのハッシュ値 |
| 保存場所 | リポジトリ内のパス |
| データ範囲の注記 | 全量か抜粋か等、データの範囲に関わる事実 |
